# Falling Block Clock

落下するブロックで時刻を表示するデジタル時計のウェブアプリケーションです。

## 特徴

- リアルタイムで更新される時刻表示
- アニメーションで数字が落下する視覚効果
- スタイリッシュなデザインと色使い
- レスポンシブデザイン

## 使用方法

1. `index.html` をウェブブラウザで開くだけで使用できます
2. 時、分、秒が落下するブロックのアニメーションで表示されます

## 技術スタック

- HTML5
- CSS3（アニメーション）
- JavaScript（DOM操作）
